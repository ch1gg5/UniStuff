## 📌 How This Folder Is Organised

Each module has its own sub-folder named after the module code and/or title.  
Inside each folder you will find annotated source files that explain **what the code does and why**.

```
Practical-Lecture-Notes/
├── <Module-Code>-<Module-Name>/
│   ├── README.md          ← Module overview and topics covered
│   ├── topic-1.py         ← Annotated code snippet (language varies)
│   ├── topic-2.py
│   └── ...
└── Example-Module/        ← Template showing the expected structure
```
