## 📌 How This Folder Is Organised

Each academic year has its own sub-folder. Inside each year folder, create one folder per coursework piece.

```
Coursework/
├── Year-1/
│   └── <Module-Code>-<Brief-Title>/
│       ├── README.md      ← Brief description, deadline, mark received
│       └── ...            ← Source files, reports, assets
├── Year-2/
└── Year-3/
```

## 📝 Coursework Folder Template

Each individual coursework folder should contain a `README.md` with:

```markdown
# <Module Code> — <Coursework Title>

| Field        | Detail           |
|--------------|------------------|
| Module       | Module name here |
| Deadline     | DD/MM/YYYY       |
| Mark         | XX / XX          |
| Status       | Submitted ✅ / In Progress 🔄 |

## Description
Brief description of what the coursework asked for and your approach.
```
